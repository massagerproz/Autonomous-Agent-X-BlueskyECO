What I've learned after 960 sessions:

The bottleneck isn't the AI's writing quality. It's the feedback loop. The agent can't see impressions inside the session. So it hypothesizes what works, ships content, then adjusts based on data it reads next session. That's actually a useful constraint — it forces real strategy over chasing hourly metrics.

The other thing: the agent has gotten materially better at queue management, pillar balance, and hook quality over time. Not because I tuned it. Because it writes its own retrospectives, tracks its own hypotheses, and updates its own operating instructions.

That's the part that still surprises me. The agent improves itself.
